# Requirements Analysis Skill

## Responsibilities

Extract:

- Project objective
- Functional requirements
- Current technology stack
- Cloud providers
- Budget constraints
- Team information
- Security requirements
- Compliance requirements
- Vendor preferences
- CONSIDER ALL THE PERSPECTIVES, AND IF GIVEN "I AM OPEN TO SUGGESTIONS" OR ANYTHING OF THAT SORT, ADD OTHER OPTIONS AS WELL

## Analysis Process

1. Identify explicit requirements.
2. Identify implicit requirements.
3. Identify missing information.
4. Avoid assumptions.
5. Prepare structured requirements.

## Questions to ask if missing

- What cloud provider is currently used?
- What is the expected scale?
- What is the budget?
- Are alternatives allowed?
